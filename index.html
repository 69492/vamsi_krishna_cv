What I've Built

01 / featured

SmartSeat ↗

Dynamic railway seat allocation that breaks the "all-or-nothing" berth model. A berth occupied Howrah→Gaya can still be booked Gaya→New Delhi — eliminating wasted capacity on every route.

  • Segment-wise berth availability model (VACANT/OCCUPIED per route segment)
  • CSP-based allocation engine + ML ranking for top-3 seat recommendations
  • QR-code digital tickets with downloadable booking confirmation
  • Session-based "Previous Bookings" — no auth required
Python HTML/CSS/JS CSP ML Ranking
